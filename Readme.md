Python code to convert font to symbol.cod, kandat.dat and kandat2.dat for the long rail app running in pc9801.

usage in VSCode console:

via svg:
python make_kandat.py
python make_symbol_cod.py 

via ttf:
python ttf2symbol.py <----this is the best.
python ttf2kandat.py
python linettf2kandat.py <----this is the best.
