# 二、SINV1 子程式（矩陣求逆）

這個子程式是 **Gauss-Jordan 消去法**。

```fortran

C----------------------------------------------------
C 功能
C----------------------------------------------------

C 對矩陣 A 求逆
C 同時計算行列式 DET

C 使用 Gauss-Jordan 消去法

C----------------------------------------------------

C A      : 輸入矩陣
C N      : 矩陣階數
C DETE   : 行列式
C NSTOP  : 錯誤標誌
C EPS    : 判斷奇異矩陣的精度

C----------------------------------------------------

C 尋找Pivot (主元)

      BIGST=-1.0D0

C 找最大元素

      IF(BIGST-ABS(A(I,K))) 130,140,140

C----------------------------------------------------
C Gauss-Jordan 消去
C----------------------------------------------------

      PIVOT=A(L,K)

C 行列式更新
      DETE=DETE+PIVOT

      PIVI=-1.0/PIVOT
```

# 三、這個程式的數學模型

這個 Fortran 程式其實在解：

鋼軌縱向平衡微分方程

[
EA\frac{d^2 u}{dx^2}+r(x)=0
]

其中

* (u(x)) 鋼軌位移
* (r(x)) 扣件阻力

軸力

[
N(x)=EA\frac{du}{dx}
]

溫度軸力

[
N_T=EA\alpha T
]

---

離散後形成非線性方程

[
F(Y)=0
]

利用 Newton 法

[
J(Y)\Delta Y=-F(Y)
]

其中 Jacobian

[
J=\frac{\partial F}{\partial Y}
]

# 四、程式計算流程（重要）

整個程式流程其實是：

```
讀資料
   ↓
建立軌道幾何
   ↓
計算溫度軸力 PT
   ↓
猜測斷軌位置 Y
   ↓
計算軸力 P,Q
   ↓
建立非線性方程 F
   ↓
建立 Jacobian DF
   ↓
解 JH=-F
   ↓
更新 Y
   ↓
直到收斂
   ↓
輸出軸力分佈
```
